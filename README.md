## Instalasi
* Clone Repository
```
git clone https://github.com/An-Ra/Restaurant-pwa.git
```
* Install Semua Dependecy
```
npm install
```

## Pengunaan

### Build Website

1. Compress Image 
```
npm run build-image
```
2. Build Website
```
npm run build
```
3. Hasil Build Berada di Folder `dist`
### Testing
* Unit Test dengan Karma Js
```
npm run test
```
* End To End Testing dengan Codecept dan Puppeteer
```
npm run e2e
```
